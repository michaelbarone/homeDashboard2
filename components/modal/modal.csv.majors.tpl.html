CSV format explained for majors

All data is only stored within this browser session. No data will be sent or stored on any server for this application.
All Emplid fields are hashed when imported to help anonymise both student and instructor identities.

CSV example:

Emplid Grad Year Grad Term Grad Major Degree Label Grad Total GPA Year Term Name Major Total GPA Total Units Earned Transfer Units Earned
783089809 2009 2098 EDUCATION 3.91 2009 Fall Education 3.91 120 0
783089809 2009 2098 EDUCATION 3.91 2008 Spring Education 3.8 108 0
555089809 2008 Spring Biology 2.5 12 12






EmplidThe student ID or any unique key to identify a student. This id can occur multiple times in the data set, and will be grouped as the same student.
Grad YearThe year of graduation, or blank if not graduated
Grad TermThe term code of graduation, or blank if not graduated
Grad Major Degree LabelGraduating Major, or blank if not graduated
Grad Total GPAGraduating GPA, or blank if not graduated
YearThe Current year for this term
Term CodeA numeric code for each academic term where each term code is larger than the last
Term NameA human readable description of the term
MajorMajor or official program on record for this term
Total GPATotal GPA at start of this term
Total Units EarnedTotal units earned at start of this term
Transfer Units EarnedTotal Transfer units (not earned at this university) at start of this term
